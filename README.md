# HelioRing X1

A modern landing page for **HelioRing X1**, a fictional premium smart health ring built for the Helicorp Internship Web Development Challenge.

## 🚀 Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Hook Form
- Lucide React

## 📦 Getting Started

```bash
npm install
npm run dev
```

## 📁 Project Structure

```
src/
├── components/
├── hooks/
├── data/
├── services/
├── styles/
├── utils/
└── assets/
```

## ✨ Features

- Responsive Landing Page
- Dark / Light Mode
- Smooth Scroll Animation
- Newsletter Form
- AI Chatbot Widget
- SEO Optimized
- High Google Lighthouse Score
